Senior web designer & front-end developer

Hi, I'm Eugene - the one who is constantly changing web-space for the better. Connecting the power of experienced web designer and modern front-end developer I aim to create well-designed and easy-to-use web products.

I prefer make interfaces and websites followed by front-end coding. Among the main types of preferred projects, I can distinguished: websites, e-commerce, portals and web apps.
